Nossa Rota de Programação

1. Fundamentos

É a base de tudo, o "alfabeto" da programação. Antes de escrever frases complexas, você precisa entender as letras e as regras de pontuação.

O que é: A sintaxe básica da linguagem, como escrever comentários, a diferença entre letras maiúsculas e minúsculas (case-sensitivity), e como o computador lê o seu código (de cima para baixo).

Analogia: Aprender o alfabeto, os números e as regras básicas de gramática de um novo idioma.

2. Variáveis e Dados

Se programas manipulam informações, as variáveis são os potes onde guardamos essas informações para usar depois.

O que são: Variáveis são "caixas" com etiquetas (nomes) que armazenam Dados. Os principais tipos de dados (dados primitivos) são:

  • String: Texto (ex: "Olá, mundo!").
  • Number: Números, sejam inteiros ou com vírgula (ex: 25, 3.14).
  • Boolean: Um valor de verdadeiro ou falso (true ou false).

Exemplo: let idade = 25; cria uma caixa chamada idade e guarda o número 25 dentro dela.

3. Operadores

São os símbolos que usamos para fazer "coisas" com nossas variáveis e dados, como contas de matemática ou comparações.

O que são:

  • Aritméticos: Para fazer contas (+, -, *, /).
  • De Comparação: Para comparar valores (== igual, != diferente, > maior que, < menor que). O resultado é sempre um Boolean (true ou false).
  • Lógicos: Para combinar expressões de verdadeiro ou falso (&& E, || OU, ! NÃO).

Exemplo: if (idade >= 18 && temHabilitacao == true) usa operadores de comparação (>=) e lógico (&&).

4. Condições

São as "bifurcações" no nosso código. Permitem que o programa tome decisões e execute caminhos diferentes dependendo da situação.

O que são: O bloco if / else (Se / Senão). "SE (if) uma condição for verdadeira, faça isto. SENÃO (else), faça aquilo."

Analogia: Um GPS que diz: "Se houver trânsito na avenida principal, então vire à direita, senão, siga em frente."

5. Repetições (Laços ou Loops)

Permitem executar o mesmo bloco de código várias vezes, sem precisar escrevê-lo de novo e de novo. Essencial para automatizar tarefas.

O que são: Os comandos for e while.

  • for: Usado quando você sabe quantas vezes quer repetir algo. ("Repita 10 vezes").
  • while: Usado quando você quer repetir algo enquanto uma condição for verdadeira. ("Repita enquanto a porta estiver fechada").

Exemplo: Para mostrar os números de 1 a 5, em vez de 5 console.log, você usa um for.

6. Arrays (Vetores)

Em vez de criar uma variável para cada item, você pode criar uma "lista" de itens e guardar tudo numa variável só.

O que é: Uma coleção ordenada de valores. Pense numa estante com prateleiras numeradas (a partir do 0).

Exemplo: let listaDeCompras = ["maçã", "banana", "leite"];. Para pegar o primeiro item, usamos listaDeCompras[0].

7. Functions (Funções)

São blocos de código reutilizáveis que realizam uma tarefa específica. São a principal forma de organizar e estruturar um programa.

O que são: Uma "receita" de código que você nomeia e pode "chamar" (executar) sempre que precisar. Elas podem receber dados (parâmetros) e retornar um resultado (return).

Analogia: Uma máquina de café. Você coloca os "ingredientes" (pó de café e água) como parâmetros, a máquina (função) faz seu trabalho, e te entrega o "resultado" (o café pronto).

8. DOM (Document Object Model)

Essa é a ponte que conecta o seu JavaScript com o HTML da sua página. É o que permite que seu código "converse" com a página e a modifique.

O que é: Uma representação em formato de "árvore" de todos os elementos do seu HTML. O JavaScript pode usar o DOM para encontrar, alterar, adicionar ou remover qualquer elemento e seu conteúdo.

Exemplo: document.querySelector('button') usa o DOM para encontrar um botão no HTML. elemento.innerHTML = "Novo Texto" usa o DOM para mudar o que está escrito dentro de um elemento.

Juntando Tudo

Imagine o seu projeto "Analisador de Números":

Você usa o DOM para pegar o que o usuário digita no input. Com Operadores e Condições, você verifica se o número é válido. Se for, você o armazena em um Array (uma Variável especial). Tudo isso é organizado dentro de uma Função adicionar(), que é chamada quando o usuário clica no botão. Se você quisesse exibir todos os números do array na tela, usaria um laço de Repetição para percorrer o array e o DOM para criar e exibir cada item na página.
